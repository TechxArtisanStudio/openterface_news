KeyCmd **0.19** (`versionCode` **19**) — это крупное обновление, которое включает **ребрендинг приложения** с KeyMod на KeyCmd, совершенно новый **режим KM Pro Compose** с поддержкой HID-отправки с поддержкой Unicode, расширенный **многоязычный пользовательский интерфейс** (включая корейский, итальянский, русский и бразильский португальский), **интерактивные гиды для каждого режима** и десятки UX-усовершенствований клавиатуры. геймпад и режимы презентации.

## Ребрендинг приложения: KeyMod → KeyCmd

Отображаемое имя приложения теперь — **KeyCmd** во всех точках входа. Этот ребрендинг проясняет различие между **аппаратным обеспечением KeyMod** и сопутствующим ему **приложением KeyCmd**.

<img src="https://assets2.openterface.com/images/keymod/KeyCmd-Welcome-Screen-landscape.webp" alt="Экран приветствия KeyCmd" style="max-width:100%; border-radius:8px; Margin:16px 0;" loading="ленивый">

### Что изменилось

- **Отображаемое имя приложения**: на значке запуска и в системном интерфейсе теперь отображается «KeyCmd».
- **Приветствие**: обновлен текстовый знак и копия из KeyMod в KeyCmd.
- **Артефакты CI и имена файлов APK**: используйте префикс **KeyCmd**.
- `applicationId` остаётся **`com.openterface.keymod`** для плавного обновления на месте.

Существующие пользователи: ваши настройки, профили и сопряженные устройства сохраняются. Обновление происходит без проблем.

## KM Pro: режим создания и отправки

Самая большая новая функция версии 0.19 — это **режим создания** в KM Pro — специальный текстовый редактор, который позволяет вводить длинные отрывки и отправлять их в виде HID-нажатий клавиш на целевой компьютер.

### Редактор создания сообщений

- **Верхняя полоса быстрого доступа + строка действий** с элементами управления **Очистить** и **Отменить/Очистить**.
- **Сохранение черновиков**: ваш текст сохраняется при переключении подрежимов и даже после успешной отправки.
- **Интеграция с IME**: вводите текст с клавиатуры телефона, отправляйте в виде чистых нажатий HID-клавиш.
- **Определить ход отправки**: видимый индикатор выполнения для длинных HID-буферов, чтобы вы точно знали, как далеко продвинулась отправка.

### Отправка HID с поддержкой Unicode

- **Двухрежимный анализ рисков**: перед отправкой текста, отличного от ASCII, в диалоговом окне предупреждается о совместимости Unicode и предлагаются действия по проверке/предварительному просмотру.
- **Шестнадцатеричный поток Unicode для macOS**: на хостах macOS приложение использует метод ввода шестнадцатеричного кода Option+ для расширенных символов.
- **Диалоги безопасной отправки**: экран просмотра адаптирует свое содержимое в зависимости от того, является ли буфер чистым ASCII или содержит символы Юникода.
- **Инструменты проверки символов**: диалоговое окно отправки рисков предоставляет действия **Проверка** и **Предварительный просмотр**, а хосты macOS включают специальную запись **Аудит шестнадцатеричного пути в Юникоде**.

### Базовый объем КМ

В версии 0.19 **Написание и отправка остается функцией клавиатуры и мыши Pro**. KM Basic фокусируется на рабочих процессах с полноэкранной клавиатурой, тачпадом и цифровой клавиатурой.

## Многоязычная поддержка

KeyCmd теперь поддерживает **11 языков пользовательского интерфейса приложений**. В этом выпуске добавлены четыре новые локализации:

- **Корейский (ко)**: полный перевод пользовательского интерфейса.
- **Итальянский (it)**: полный перевод пользовательского интерфейса.
- **Русский (ru)**: полный перевод пользовательского интерфейса.
- **Бразильский португальский (pt-BR)**: полный перевод пользовательского интерфейса.

В сочетании с существующими английским, упрощенным китайским, традиционным китайским, японским, французским, немецким и испанским языками KeyCmd теперь охватывает подавляющее большинство нашей глобальной базы пользователей.

### Что изменилось

- **Выбор языка** в настройках с каноническими названиями языков приложений.
- **Заголовки Bluetooth** и **предварительный просмотр нажатия клавиш** локализованы.
- **Выпуск lint** исправлений для вкладок с предупреждениями о создании сообщений на всех языках.

## Интерактивные туры с гидом

Каждый режим теперь имеет **встроенную интерактивную экскурсию**, которая шаг за шагом знакомит вас с его функциями.

### Доступные туры

- **Обзор по Shortcut Hub**: открывает профиль по умолчанию и содержит подробные сведения о пользовательском интерфейсе, вкладках категорий и управлении ярлыками.
- **Тур по геймпаду**: объясняется расположение геймпада, управление модулями и система предустановок.
- **Обзор KM Pro**: рассказывается о режиме создания сообщения, панели ярлыков и функциях, специфичных для версии Pro.
- **Основной обзор KM**: поясняет работу полноэкранной клавиатуры, модификатора и цифровой клавиатуры.

### Особенности тура

- **Руководства для каждого режима**: для каждого режима предусмотрен собственный тур.
- **Лист воспроизведения**: в любое время можно вернуться к любому туру с помощью кнопки **Справочник по режимам** (значок слева от хромированной панели подключения).
- **Поддержка i18n**: содержимое тура локализовано для всего набора языков приложения.
- **Оптимизация для альбомной ориентации**: макеты нижних листов правильно адаптируются в альбомной ориентации.

## Усовершенствования UX

### Клавиатура

- **Предварительный просмотр нажатия клавиши**: посмотрите, что именно будет отправлено, прежде чем нажать. Включить через настройки. Включено по умолчанию.
- **Исправление HID при быстром нажатии**: улучшено время отпускания нажатия на клавиатуру и объединены события отпускания для более быстрого набора текста.
- **Альтернативное управление касанием**: при длительном нажатии клавиши «a» теперь отображаются альтернативы ¥ (вверх), £ (влево), € (вправо) с улучшенной визуальной обратной связью.
- **Проведение с удерживанием модификатора**: в турах KM Basic/Pro новый шаг обучает жесту удержания смахивания для быстрого доступа к модификатору.

### Геймпад

- **Панель редактирования сеанса удалена**: более чистый хром геймпада без старой панели инструментов редактирования сеанса.
- **Геймпад Chrome и тур**: новая визуальная обработка и встроенный гид-экскурсия.
- **Значок руководства по режимам**: расположен слева от хромированной панели подключения для облегчения доступа.

### Презентация

- **Блокировка портретной пары**: режим презентации ограничен книжной и обратной книжной ориентацией для стабильного управления докладчиком.

### Пользовательский интерфейс и тема

- **Образцы акцентов**: счетчик семейства цветов темы заменен образцами визуальных акцентов для упрощения выбора цвета.
- **Выравнивание акцентов пользовательского интерфейса**: все акценты пользовательского интерфейса теперь соответствуют семейству цветов темы (а не устаревшему основному цвету).
- **Кластер справа в заголовке**: улучшены размеры значков подключений как в главном приложении, так и в Chrome KM Basic.
- **Стиль кнопки отправки сообщения**: выровнен внешний вид кнопки отправки, отличной от ASCII, в облегченном режиме.

### Настройки

- **Реорганизация настроек**: канонические названия языков приложений; переименованы скрипты установки эмулятора для ясности
- **Вспомогательные сценарии для разработчиков**: переименованы для более четкой идентификации устройства и именования действий.
- **Документация часто задаваемых вопросов**: обновлен `docs/FAQ.md` с учетом текущего поведения приложения.

KeyCmd предназначен не только для управления сервером. Разработчики используют его для **удаленных сеансов кодирования** — управления безголовым компьютером с телефона или планшета с полной поддержкой клавиатуры, тачпада и цифровой панели.

## Работает с Mini-KVM и KVM-Go.

Приложение KeyCmd не ограничивается оборудованием KeyMod. Существующие пользователи Openinterface также могут попробовать его:

- **KVM-Go**: подключение через **Bluetooth** или **USB**
- **Мини-KVM**: подключение через **USB**

## Примечания к обновлению

- **Ребрендинг**: название приложения меняется с KeyMod на KeyCmd. Ваши данные сохраняются.
