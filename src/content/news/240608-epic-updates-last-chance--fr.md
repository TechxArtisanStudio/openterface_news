Salut tout le monde !

Le temps passe vite quand on s'amuse ! Nous sommes maintenant dans la dernière semaine de notre campagne de financement participatif pour l'Openterface Mini-KVM sur Crowd Supply. Plongeons dans quelques mises à jour passionnantes !

## Mises à jour Logiciel

Nous avons une excellente nouvelle pour vous – notre mini-KVM supporte maintenant macOS, Windows et Linux ! De plus, tout est open source !

🎉 Consultez les détails pour chaque système ci-dessous :

### Openterface_MacOS

  - **Dépôt GitHub :** [Openterface_QT](https://github.com/TechxArtisanStudio/Openterface_QT)
  ![openterface-macos-app-store](https://www.crowdsupply.com/img/50cb/9cdf2fb2-d3e9-411c-a90e-9fb2e1ac50cb/openterface-macos-app-store-1_png_gallery-lg.jpg)
  - **Vidéo Démo Bêta Précoce :** [Mini-KVM : Tests de Contrôle KVM de Base](https://www.youtube.com/watch?v=m7OpUem0zqY)
  - **Comment Installer :**
    - **App Store :** Trouvez notre app hôte en recherchant "Openterface" ou "Mini-KVM".
    - **Construire depuis la Source :** Installez CodeX et construisez l'app macOS depuis la source.

### Openterface_QT

  - **Dépôt GitHub :** [Openterface_QT](https://github.com/TechxArtisanStudio/Openterface_QT)
  - **Démos Bêta Précoces :**
    - **Windows :** [Mini-KVM : Vidéo Démo Bêta pour Windows](https://www.youtube.com/watch?v=ERzpGtRvP2o&t=23s)
    ![openterface-win-app-demo-2](https://www.crowdsupply.com/img/d146/26c5df78-f942-4743-ad32-97659a89d146/openterface-win-app-demo-2-1_jpg_gallery-lg.jpg)

    - Téléchargez l'app Windows depuis notre release GitHub [ici](https://github.com/TechxArtisanStudio/Openterface_QT/releases/tag/v0.0.1).
    - Construisez depuis la source pour Windows et Linux en suivant nos instructions.
    - **Défi Raspberry Pi :** Nous travaillons encore dur pour faire fonctionner notre app hôte en douceur sur Raspberry Pi. Nous y arriverons bientôt.

### Autres Apps

  - Comme mentionné dans notre [mise à jour précédente](/blog/from-development-to-your-hands--behind-the-scenes-/#openterface_android-and-openterface_webextension), nous avons terminé les preuves de concept pour Android et WebExtension. Bien qu'elles soient de priorité plus faible, elles sont définitivement dans le pipeline. Nous explorons aussi le support ChromeOS et la possibilité de faire fonctionner le mini-KVM sur les iPad avec puces de série M. Rejoignez la discussion dans notre communauté si vous avez des idées !

### Sac Trousse à Outils Amélioré

Avec le nouvel adaptateur pratique attaché pour notre câble Type-C 1,5m et la [mise à jour du câble VGA-to-HDMI de 1m](/blog/-upgrade-on-vga-to-hdmi-cable-as-a-free-bonus-/), nous pensons qu'il est nécessaire d'augmenter la taille de notre sac trousse à outils à 16 cm L x 10 cm W x 3,8 cm H, offrant un peu plus d'espace !
