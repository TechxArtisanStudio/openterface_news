**2026년 6월 22일** — uConsole KVM 확장의 차세대 버전이 출시되었습니다. **Openterface uConsole KVM Extension Module v2**가 [TechxArtisan Shop](https://shop.techxartisan.com/products/openterface-kvm-ext-for-uconsole)에서 배송을 시작했습니다.

![kvmext-v2-hero-1](https://assets.openterface.com/images/product/kvmext-v2/kvmext-v2-hero-1.webp)

## v2의 새로운 기능

Extension Module v2는 Clockwork uConsole을 단일 확장 슬롯만으로 **휴대용 KVM 콘솔**, **네트워크 디버그 터미널**, **SD 카드 유틸리티**로 변환합니다:

| 기능 | v1 (기존) | **v2 (현재 출시)** |
|---------|-------------|------------------------|
| 보드 크기 | 37 × 77 mm | **77.3 × 34.7 mm** |
| 이더넷 | — | **100M + 1000M 카드 포함** |
| SD 카드 | — | **호스트/타겟 전환 지원 읽기/쓰기** |
| 비디오 출력 | 1080p @ 30 Hz, &lt;140 ms | **1080p @ 60 Hz, &lt;70 ms** |
| USB | 전환 가능한 USB 2.0 | 전환 가능한 USB 2.0 |

## 단일 SKU — 두 네트워크 카드 모두 포함

모든 제품에는 **100M과 1000M 이더넷 카드가 모두** 포함되어 있습니다. 모든 uConsole 기본 보드에서는 100M을 사용하세요. **HackerGadgets uConsole Upgrade Kit**(결제 시 번들 옵션으로 제공)를 보유한 경우 1000M을 사용하세요.

## 구매 옵션

1. **Extension v2 단품** — 모듈 + 두 네트워크 카드 + 슬롯 커버
2. **Extension v2 + Upgrade Kit 번들** — 위의 모든 항목과 기가비트 지원 설정을 위한 HackerGadgets Upgrade Kit
