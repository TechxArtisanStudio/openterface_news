Привет всем!

Время летит незаметно, когда тебе весело! Сейчас идет последняя неделя нашей краудфандинговой кампании для Openterface Mini-KVM on Crowd Supply. Давайте погрузимся в некоторые интересные обновления!

## Обновления программного обеспечения

У нас для вас потрясающие новости: наш мини-KVM теперь поддерживает macOS, Windows и Linux! Более того, все это с открытым исходным кодом!

🎉 Ознакомьтесь с подробностями каждой системы ниже:

### Openinterface_MacOS

  - **Репозиторий GitHub:** [Openterface_QT](https://github.com/TechxArtisanStudio/Openterface_QT)
  ![openterface-macos-app-store](https://www.crowdsupply.com/img/50cb/9cdf2fb2-d3e9-411c-a90e-9fb2e1ac50cb/openterface-macos-app-store-1_png_gallery-lg.jpg)
  - **Демо-видео ранней бета-версии:** [Mini-KVM: Basic KVM Control Tests](https://www.youtube.com/watch?v=m7OpUem0zqY)
  - **Как установить:**
    - **App Store:** найдите наше хост-приложение, выполнив поиск «Openterface» или «Mini-KVM».
    - **Сборка из исходного кода.** Установите CodeX и создайте приложение для macOS из исходного кода.

### Openinterface_QT

  - **Репозиторий GitHub:** [Openterface_QT](https://github.com/TechxArtisanStudio/Openterface_QT)
  - **Ранние бета-демоверсии:**
    - **Windows:** [Mini-KVM: Beta Demo Video for Windows](https://www.youtube.com/watch?v=ERzpGtRvP2o&t=23s)
    ![openterface-win-app-demo-2](https://www.crowdsupply.com/img/d146/26c5df78-f942-4743-ad32-97659a89d146/openterface-win-app-demo-2-1_jpg_gallery-lg.jpg)
    - **Linux:** [Mini-KVM: Beta Demo Video for Linux](https://www.youtube.com/watch?v=_ScpI6TC0Pk)
    ![openterface-linux-app-demo.png](https://www.crowdsupply.com/img/61a9/58109b24-3d4e-4058-8377-9860631661a9/openterface-linux-app-demo_png_md-xl.jpg)
  - **Как установить:**
    - Загрузите приложение для Windows из нашей версии GitHub [here](https://github.com/TechxArtisanStudio/Openterface_QT/releases/tag/v0.0.1).
    - Соберите исходный код для Windows и Linux, следуя нашим инструкциям.
    - **Raspberry Pi Challenge:** Мы все еще прилагаем все усилия, чтобы обеспечить бесперебойную работу нашего хост-приложения на Raspberry Pi. Мы просто скоро доберемся туда.

### Другие приложения

  - Как упоминалось в нашем [previous update](/blog/from-development-to-your-hands--behind-the-scenes-/#openterface_android-and-openterface_webextension), мы завершили проверку концепции для Android и WebExtension. Хотя они имеют более низкий приоритет, они определенно находятся в стадии разработки. Мы также изучаем поддержку ChromeOS и возможность работы мини-KVM на iPad с чипами серии M. ## Еще лучшие аксессуары

### НОВЫЙ оранжевый кабель Type-C длиной 1,5 м с преобразователем

![type-c-cable-v2-plugged-old-mac](https://www.crowdsupply.com/img/9871/2f6f967e-b9ea-4b48-b5dd-da135fb29871/type-c-cable-v2-plugged-old-mac_jpg_md-xl.jpg)

Помните наш [stylish orange Type-C cable](/blog/from-development-to-your-hands--behind-the-scenes-/#upgrading-toolkit-accessories) из чудесного силикона? Мы получили первые образцы, и они выглядят потрясающе! Эти кабели поддерживают **быструю зарядку 240 Вт** (напряжение постоянного тока 50 В, ток 5 А, мощность 240 Вт) и идеально работают с нашими мини-KVM. Большое спасибо нашему производителю и нашим спонсорам за то, что это произошло!

![type-c-cable-v2-open-2](https://www.crowdsupply.com/img/71b2/b37b66e3-7f2e-4c5e-bb45-8944ee2971b2/type-c-cable-v2-open-2_jpg_gallery-lg.jpg)


### Улучшенная сумка для инструментов

Мы считаем, что с новым удобным адаптером для кабеля Type-C длиной 1,5 м и [1m long VGA-to-HDMI cable update](/blog/-upgrade-on-vga-to-hdmi-cable-as-a-free-bonus-/) необходимо увеличить размер нашей сумки для инструментов до 16 см Д x 10 см Ш x 3,8 см В, предоставив немного больше места!

![beta-round2-toolkit](https://www.crowdsupply.com/img/0f20/4aed395b-dbef-4670-b340-403ee8e30f20/beta-round2-toolkit_jpg_md-xl.jpg)
