**June 22, 2026** — The next generation of our uConsole KVM expansion is here. **Openterface uConsole KVM Extension Module v2** is now shipping from the [TechxArtisan Shop](https://shop.techxartisan.com/products/openterface-kvm-ext-for-uconsole).

![kvmext-v2-hero-1](https://assets.openterface.com/images/product/kvmext-v2/kvmext-v2-hero-1.webp)

## What's new in v2

Extension Module v2 turns your Clockwork uConsole into a **portable KVM console**, **network debug terminal**, and **SD card utility** — all from one expansion slot:

| Feature | v1 (legacy) | **v2 (now shipping)** |
|---------|-------------|------------------------|
| Board size | 37 × 77 mm | **77.3 × 34.7 mm** |
| Ethernet | — | **100M + 1000M cards included** |
| SD card | — | **Read/write with host/target switching** |
| Video output | 1080p @ 30 Hz, &lt;140 ms | **1080p @ 60 Hz, &lt;70 ms** |
| USB | Switchable USB 2.0 | Switchable USB 2.0 |

## Single SKU — both network cards in the box

Every unit includes **both** the 100M and 1000M Ethernet cards. Use 100M on all uConsole base boards. Use 1000M when you have the **HackerGadgets uConsole Upgrade Kit** (available as a bundle option at checkout).

## Purchase options

1. **Extension v2 Only** — module + both network cards + slot cover
2. **Extension v2 + Upgrade Kit Bundle** — everything above plus the HackerGadgets Upgrade Kit for gigabit-ready setup
